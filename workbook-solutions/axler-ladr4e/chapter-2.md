### Removing vector from list of linearly independent

page 33

$ a_1 v_1 + a_2 v_2 + ... + a_m v_m = 0 \ \Leftrightarrow \ a_1 = 0, \ a_2 = 0, \ ..., \ a_m = 0 $

Imagine

$ a_1' v_1 + a_2' v_2 + ... + a_{m-1}' v_{m-1} = 0 $

Some $ a_k' \ne 0 $

Then

$ a_1' v_1 + a_2' v_2 + ... + a_{m-1}' v_{m-1} + 0 \ v_m  = 0 $

And some $ a_k' \ne 0 $
